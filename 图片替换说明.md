# 图片替换说明

## 📋 两种替换方案

### 方案一：直接替换 PNG（最简单，推荐）✨

**操作步骤**：

1. **优化图片**：
   - 使用 Squoosh (https://squoosh.app/) 压缩原 PNG 图片
   - 保持 PNG 格式（不需要转换为 WebP）
   - 调整质量到 75-85，确保视觉质量

2. **直接替换文件**：
   - 将优化后的 `wallImage.png` 直接放到：`C:\Users\Surface\Desktop\mecollage\public\wallImage.png`
   - 将优化后的 `placeholder-icon.png` 直接放到：`C:\Users\Surface\Desktop\mecollage\public\placeholder-icon.png`
   - 将优化后的 `upload-icon.png` 直接放到：`C:\Users\Surface\Desktop\mecollage\public\upload-icon.png`
   - **直接覆盖原文件即可**（不需要删除，直接替换）

3. **命名规则**：
   - ✅ **必须保持原命名**：`wallImage.png`、`placeholder-icon.png`、`upload-icon.png`
   - ✅ 代码中已经引用了这些文件名，如果改名需要更新代码

4. **upload-icon.png 特殊处理**：
   - 需要调整尺寸为 **66x72 像素**（或 132x144 for 2x 高DPI屏幕）
   - 在 Squoosh 中可以调整尺寸

**优点**：
- ✅ 最简单，不需要修改代码
- ✅ 直接替换即可
- ✅ 所有浏览器都支持

**缺点**：
- ⚠️ PNG 文件可能仍然比 WebP 稍大（但已经压缩后差异不大）

---

### 方案二：使用 WebP 格式（更优，但需要更新代码）

**操作步骤**：

1. **优化并转换为 WebP**：
   - 使用 Squoosh 转换为 WebP 格式
   - 质量设置为 75-85

2. **文件放置**：
   - 将 `wallImage.webp` 放到：`C:\Users\Surface\Desktop\mecollage\public\wallImage.webp`
   - 将 `placeholder-icon.webp` 放到：`C:\Users\Surface\Desktop\mecollage\public\placeholder-icon.webp`
   - 将 `upload-icon.webp` 放到：`C:\Users\Surface\Desktop\mecollage\public\upload-icon.webp`
   - **保留原 PNG 文件作为回退**（不删除）

3. **更新代码**（我会帮您完成）：
   - 更新 CSS 和 HTML 以支持 WebP，并提供 PNG 回退

**优点**：
- ✅ 文件更小（比 PNG 小 25-35%）
- ✅ 更好的性能

**缺点**：
- ⚠️ 需要更新代码
- ⚠️ 需要保留两套文件（WebP + PNG 回退）

---

## 🎯 推荐方案

**推荐使用方案一（直接替换 PNG）**，因为：
1. 最简单，不需要修改代码
2. 压缩后的 PNG 已经能大幅减少文件大小
3. 所有浏览器都支持，无需回退

## 📝 具体操作步骤（方案一）

### 1. 优化 wallImage.png

1. 打开 https://squoosh.app/
2. 上传 `C:\Users\Surface\Desktop\mecollage\public\wallImage.png`
3. 选择 **PNG** 格式（或尝试 **MozJPEG** 如果原图是照片）
4. 调整质量滑块，直到文件大小 < 200 KiB（同时保持视觉质量）
5. 点击下载
6. 将下载的文件**重命名为 `wallImage.png`**
7. 直接拖拽到 `C:\Users\Surface\Desktop\mecollage\public\` 文件夹，**覆盖原文件**

### 2. 优化 placeholder-icon.png

1. 同样使用 Squoosh
2. 优化到 < 200 KiB
3. 下载后重命名为 `placeholder-icon.png`
4. 覆盖 `C:\Users\Surface\Desktop\mecollage\public\placeholder-icon.png`

### 3. 优化 upload-icon.png（需要调整尺寸）

1. 使用 Squoosh
2. **先调整尺寸**：
   - 点击 "Resize" 选项
   - 设置为 **132x144** 像素（2x 分辨率，适配高DPI屏幕）
   - 或 **66x72** 像素（1x 分辨率）
3. 然后优化质量，目标 < 20 KiB
4. 下载后重命名为 `upload-icon.png`
5. 覆盖 `C:\Users\Surface\Desktop\mecollage\public\upload-icon.png`

替换完成后：
