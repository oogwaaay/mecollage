# ⚡ 立即执行 - 3 步完成部署

## 🎯 最简单的方法：使用 GitHub Desktop

### 第 1 步：打开 GitHub Desktop（2 分钟）

1. **如果已安装 GitHub Desktop**：
   - 打开 GitHub Desktop
   - 点击 **File** → **Add Local Repository**
   - 浏览到：`C:\Users\Surface\Desktop\mecollage`
   - 点击 **Add repository**

2. **如果还没安装**：
   - 访问：https://desktop.github.com/
   - 下载并安装（约 5 分钟）
   - 登录您的 GitHub 账号
   - 然后按上面的步骤添加仓库

### 第 2 步：提交并推送（1 分钟）

在 GitHub Desktop 中：

1. **查看更改**：左侧会显示所有更改的文件（约 10+ 个文件）
2. **输入提交信息**（在底部）：
   ```
   Add SEO optimizations, performance improvements, and GitHub Actions auto-deployment
   ```
3. **点击 "Commit to main"**（或您的主分支名称）
4. **点击 "Push origin"** 按钮（右上角）

### 第 3 步：启用自动部署（1 分钟）

1. **打开浏览器**，访问您的 GitHub 仓库
2. **点击 Settings**（仓库设置）
3. **点击 Pages**（左侧菜单）
4. **在 "Source" 下拉菜单中选择：GitHub Actions**
5. **点击 Save**

### ✅ 完成！

等待 2-3 分钟，GitHub Actions 会自动：
- 构建项目
- 部署到 GitHub Pages
- 更新网站

**查看进度**：在仓库页面点击 **Actions** 标签，您会看到 "Build and Deploy" 工作流正在运行。

## 🆘 如果 GitHub Desktop 不可用

### 备选方案：使用 VS Code

1. 在 VS Code 中打开 `mecollage` 文件夹
2. 点击左侧 **Source Control** 图标（或按 `Ctrl+Shift+G`）
3. 输入提交信息并提交
4. 点击 **...** → **Push**

### 或者：使用 Git Bash

1. 右键点击 `mecollage` 文件夹
2. 选择 **Git Bash Here**
3. 运行：
   ```bash
   git add .
   git commit -m "Add SEO optimizations, performance improvements, and GitHub Actions auto-deployment"
   git push
   ```
