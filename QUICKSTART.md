# Quick Start Guide

## 快速开始

1. **安装依赖**
   ```bash
   cd PicStitch
   npm install
   ```

2. **启动开发服务器**
   ```bash
   npm run dev
   ```

3. **打开浏览器**
   访问 `http://localhost:3000`

## 功能说明

### 上传图片
- 拖拽图片到上传区域，或点击上传区域选择文件
- 支持同时上传多张图片
- 支持常见图片格式（JPG, PNG, GIF, WebP等）

### 选择模板
- **横向布局**: 所有图片横向排列
- **纵向布局**: 所有图片纵向排列
- **网格布局**: 多种预设网格（2x2, 3x3, 4x4, 5x5, 6x6等）
- **大网格**: 支持10x2, 20x1等大网格布局

### 自定义网格
- 可以自定义行数和列数（最多30x30）
- 适合拼接20+张图片

### 设置选项
- **间距**: 调整图片之间的间距（0-20px）
- **边框**: 添加图片边框（0-10px）
- **边框颜色**: 自定义边框颜色
- **背景颜色**: 自定义背景颜色

### 导出
- 支持PNG和JPG格式
- 可调节导出质量（10%-100%）
- 点击"Export Collage"按钮下载

## 使用技巧

1. **大量图片拼接**: 
   - 使用自定义网格功能，例如20张图片可以用4x5或5x4的网格
   - 或者使用10x2、20x1等长条形布局

2. **最佳实践**:
   - 建议图片尺寸相近，效果更好
   - 导出前可以调整间距和边框，让拼图更美观

3. **性能优化**:
   - 如果图片很多很大，导出时可能需要一些时间
   - 可以适当降低导出质量以加快速度
